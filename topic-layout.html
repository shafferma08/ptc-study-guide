The CSS Box Model

High Priority

Every HTML element is a box made of four layers. Understanding the order is critical.

  • Content: The text or image.
  • Padding: Space between content and border. (Inside)
  • Border: Outline around padding.
  • Margin: Space outside the element. (Outside)

Absolute Positioning is NOT part of the box model layers.

Content
Border
Padding
Margin (Outside)

box-sizing: border-box

The solution to "growing" elements. It includes padding and border in the element's total width.

Why it matters:

  • Keeps layouts predictable.
  • Prevents horizontal scrollbars unexpectedly.
* {
  box-sizing: border-box;
}

CSS Grid: Defining Columns

Use grid-template-columns to define the layout of columns in a grid container.

❌ Common Confusion

  • grid-columns (Not a property)
  • column-count (For multi-column text)
.container {
  display: grid;
  /* Creates 3 equal columns */
  grid-template-columns: 1fr 1fr 1fr;
}

Ordering & Sibling Selectors

Selecting elements based on their position.

Adjacent Sibling (+)

Selects the element immediately after another.

div + p
Matched: Immediate sibling only

General Sibling (~)

Selects any element after another (at same level).

div ~ p
Matched: All following siblings
