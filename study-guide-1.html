1Ô∏è‚É£ The CSS Box Model

High Priority

Every HTML element is a box made of four layers. Understanding the order is critical.

  • Content: The text or image.
  • Padding: Space between content and border. (Inside)
  • Border: Outline around padding.
  • Margin: Space outside the element. (Outside)
‚ùó Common Confusion

Margin is outside. Padding is inside.

2Ô∏è‚É£ box-sizing: border-box

The solution to "growing" elements. It includes padding and border in the element's total width.

Why it matters:

  • Keeps layouts predictable.
  • Prevents horizontal scrollbars unexpectedly.
  • Standard in modern CSS resets.
* {
  box-sizing: border-box;
}
