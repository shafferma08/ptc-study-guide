HTML Forms: GET vs POST

The method attribute controls how form data is sent.

GET

Data visible in URL. Used for searches/filtering.

POST

Data in request body. Secure. Used for passwords/submits.

Where the Data Goes

The action attribute defines the destination URL where the form submits its data.
