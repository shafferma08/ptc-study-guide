Document Structure Basics

Markup Tags: Instructions that tell the browser how to display the page content.

  • <html>
    Root element. Contains the entire webpage.
  • <body>
    Contains all visible page content. 
<html>
  <head>
    <!-- Meta info -->
  </head>
  <body>
    <!-- Visible Stuff -->
  </body>
</html>

What CSS Stands For

Cascading Style Sheets

Why "Cascading"? Because styles override each other based on priority rules (Specificity + Order).

External Stylesheets

A separate .css file controlling styles for multiple pages.

Benefits:

  • Cleaner HTML
  • Reusable Code
  • Easier Maintenance

Correct Syntax:

<link rel="stylesheet" href="style.css">

CSS Specificity & Overrides

Scenario: Parent <div> is blue. Child <p> is red.

div { color: blue; }
p { color: red; }
Result: I am Red.

Why?

  • Direct styles override inherited styles.
  • More specific selector wins.
